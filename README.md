# 🐾 Proyecto de Migración de Backend: Gestión de Mascotas

Mini proyecto desarrollado en arquitectura por capas con **Node.js, TypeScript, Express y Prisma ORM (usando PostgreSQL)** para migrar el backend del sistema de gestión de mascotas e integrarlo con un cliente en **React**.

Mini proyecto desarrollado en arquitectura por capas, ya teniendo el Frontend con React se procedio a integrar la gestión de mascota bajo la migracion de ahora el backend con **Node.js, TypeScript, Express y Prisma ORM (usando PostgreSQL)**

Con este proyecto se realiza la implementacion de un CRUD completo, adicionando los solicitados en el taller a través de la entidad de **Mascotas (Pets)**, el manejo e ingesta de **imágenes binarias (BLOB)** usando `Multer`, y la seguridad de endpoints mediante **autenticación con tokens JWT y hashing de contraseñas con Bcrypt**.

## 📁 Estructura del Repositorio

* **`Backend/`**: API REST en Node.js, Express, TypeScript y Prisma ORM.
* **`Frontend/`**: Interfaz de usuario en React + Vite.
* **`Entregable/`**: Capturas de pantalla solicitadas para la evaluación del taller.
